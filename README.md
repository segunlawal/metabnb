# MetaBnB

Website for a freelance writer

### LIVE LINK

https://segunlawal-metabnb.netlify.app/

## Tech Stack

React, TailwindCSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/segunlawal/metabnb.git
```

Go to the project directory

```bash
  cd metabnb
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
